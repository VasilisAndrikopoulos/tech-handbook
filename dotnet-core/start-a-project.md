# C# Dev Kit and SDK
Make sure that you already have installed .NET SDK from microsoft and C# Dev Kit from the Visual Studio Marketplace.

# Create an new Project
To get started, go to the Explorer view and select Create .NET Project. Alternatively, you can bring up the Command Palette using ```⇧⌘P``` and then type ".NET" and find and select the .NET: New Project command.

1. After selecting the command, you'll need to choose the project template.
2. To run your app, select Run > Run without Debugging in the upper menu, or use the ⌃F5 keyboard shortcut. To learn more about debugging your C# project, read the debugging documentation.
